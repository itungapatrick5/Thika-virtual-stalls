# Mobile API Consumer App

A simple Android application built with Jetpack Compose that consumes a public REST API to display a list of users.

## Features
- Fetches data from [JSONPlaceholder](https://jsonplaceholder.typicode.com/) using **Retrofit**.
- Parses JSON data using **Gson**.
- Displays data in a scrollable list using **Jetpack Compose** (`LazyColumn`).
- Follows modern Android development practices with **ViewModel** and **StateFlow**.
- Handles network operations asynchronously using **Kotlin Coroutines**.

## Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Networking**: Retrofit & OkHttp
- **JSON Parsing**: Gson
- **Architecture**: MVVM (Model-View-ViewModel)
